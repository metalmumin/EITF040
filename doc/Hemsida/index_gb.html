Guitar tuner

Digitala projekt, EITF40
Av Jakub Gorski (D07) och Patrik Thoresson (D07)

 

Home | Report | Source | Pictures | Swedish

 

This project is a part of the course Digital Projects (EITF040), given
at Sweden's Faculty of Engineering, LTH in Lund. This project focuses
on constructing a guitar tuner with an ATmega16. Thetuner will be able
to tune any electric instrument that can be connected to the tuner using
a 3.5mm analog jack. Seven LED diodes will indicate the offset of the
instrument's string. The difference between this tuner and other tuners
is that it will use the ATmega16â€™s internal ADC to measure the frequencies,
whereas usually Voltage-Controlled Oscillators are used.
